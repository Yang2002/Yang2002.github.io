Prob D. 数论

Solved by YangDavid at 01:25 (+)

Description

记录 $\displaystyle A(n) = \underbrace {11111\ldots1}_{n\text{ ones}}$ ，问有多少个二元组 $(i,j),1\leq i\leq n,1\leq j\leq m$ ，满足：
$$
A(i^j) \equiv 0 \pmod P
$$
其中 $P$ 是质数。数据范围：$n,m,P\leq 10^9$

Solution

$P=2,5$ 时显然是 0.

$P=3$ 时，答案是 $\displaystyle\lfloor \frac n3 \rfloor \cdot m$

考虑其它情况， 10 与 P 互质。由于 $\displaystyle A(n) = \frac{10^n - 1}{9}$ ，由此 $A(x) \equiv 0 \pmod P$ 等价于：
$$
10^n \equiv 1 \pmod P
$$
枚举 $\varphi (P)$ 的因子求出 $10 \pmod p$ 的阶 $d$ （即最小的 $d$ ，使得 $10^d \equiv 1 \pmod P$），那么此问题相当于统计有多少 $(i,j)$ 使得 $i^j\equiv 0 \pmod d$ 。$j>30$ 时的 i 的个数一定相同，直接枚举 $d$ 的因数算一下就好了。

时间复杂度 $O(\sqrt P + \omega(P-1)\log P)$

Description

Prob I. 构造，dp

Upsolved by YangDavid (+)

Description

给你长度为 $n-2$ 的数列 $b_2,b_3,\ldots,b_{n-1}$ ，要你构造出一个长度为 $n$ 的数列 $a_1,a_2,\ldots,a_n$ 满足：对于任意 $i\in \{2,3,\ldots,n-1\}$ ，都有 $b_i = \operatorname {mid}(a_{i-1},a_i,a_{i+1})$ ，其中 $\operatorname {mid}$ 表示中位数。如果无解，则输出 -1。

Solution

这题一开始的思路就有问题，我以为可以通过构造性算法解出来呢，于是讨论了很多种情况，仍然觉得没有解决问题。事实上只需要注意下面一点：

如果有解，那么必然有一个解满足 $a_i$ 的取值为 $b_{i-1},b_i,b_{i+1}$ 中的一个。

证明比较显然。于是就可以 dp 了。设 dp[i][j][k] 表示考虑完前 $i$ 个数的取值，且 $a_{i-1}=b_{i-1+j-1},a_i = b_{i+k-1}$ 是否可行，转移是显然的。
