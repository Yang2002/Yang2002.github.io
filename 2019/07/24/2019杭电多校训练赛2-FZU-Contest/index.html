Prob G. 不平等博弈，Surreal Number

Upsolved by YangDavid

Description

给了多个 $3\times 3$ 棋盘， Alice 和 Bob 轮流选择棋盘在上面进行操作，两人的操作不同。不能操作的人会输，问最后谁能赢。

Solution

这道题就是不平等博弈的模板题。与平等博弈的 SG 函数类似，不平等博弈局面的值采用 Surreal Number 来描述。Surreal Number 组成最大的有序域，具体性质可以参考高德纳的《研究之美》进行学习，这里仅仅对算法竞赛中常用到的 Surreal Number 的性质加以总结。

公理1 任何一个 Surreal Number 均可定义为：
$$
G = \{G_L\mid G_R\}
$$
其中 $G_L,G_R$ 是已经定义好的 Surreal Number 组成的两个集合，满足不存在 $l\in G_L,r\in G_R,l \geq r$。对应于博弈之中， $G_L$ 表示 Alice 行动之后的状态对应的超现实数的集合， $G_R$ 表示 Bob 行动之后的状态对应的超现实数的集合。而要求的条件也很重要，如果满足这条公理，则可以完全按照下面的简化规则和判定规则解决基本所有问题，否则需要讨论很多拓展情况。

公理2
$$
x\leq y \Leftrightarrow \forall l\in X_L, l \ngeq y \text{ 且 }\forall r\in Y_R, x \ngeq r
$$

简化规则：事实上，记 $l_m = \max L, r_m = \min R$ ，则 $G$ 一定和 $\displaystyle \Big\{\{l_m\} \mid \{r_m\}\Big\}$ 等价，称后者为最简数字。为了处理超现实数，我们肯定要找到一套规则把它们当作实数来处理，下面是几条规则：
$$
\begin{aligned}
\{n\mid \} &= n + 1 \\
\{\mid n\} &= n - 1 \\
\{x\mid y\} &= 0 \text{ if } x < 0 < y
\end{aligned}
$$
最重要的求超现实数的规则在这里：如果 L 和 R 之间有整数，那么为最靠近 0 的整数；否则为分母为 2 的幂次且分母最小的分数，下面是一种参考实现：(其中 ceil(x) 表示严格大于 x 的最小整数，floor(x) 表示严格小于 x 的最大整数，Sur 是超现实数类，用分数实现，x 是分子，y 是分母，up() 表示乘以 2，down(d) 表示除以 $2^d$)

判定规则：如果超现实数大于 0 ，则说明 Alice 必胜；如果超现实数等于 0 ，则说明后手必胜；如果超现实数小于 0 ，则说明 Bob 必胜。多个并行游戏的值即为这几个游戏分别对应的超现实数之和。

由此，不平等博弈暴力 dp 方法的大致框架就是这样：

  • 找到两名玩家在状态 $S$ 进行操作分别可以转移到的状态集合 $S_L, S_R$
  • 算出 $S_L$ 的最大值,$S_R$ 的最小值，得到最简数字 $\{s_L\mid s_R\}$
  • 按照上面的方法，找到 $\{s_L\mid s_R\}$ 对应的分数，作为状态 $S$ 的 dp 值。

对于这道题，模拟上述过程即可，状压 dp 求出所有棋盘对应的 Surreal Number ，然后加一下就得到了并行棋局的答案，和 0 比较大小即可。
